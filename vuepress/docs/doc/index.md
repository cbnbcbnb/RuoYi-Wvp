# 介绍

ruoyi-wvp 是基于ruoyi-vue和wvp框架开发，开箱即用的28181协议视频平台。

# 1.技术栈

- 前端：vue3、element plus、axios
- 后端：springboot、mybatis、spring security、redis、jwt
- 数据库：mysql
- 视频服务器：ZLMediaKit

- Maven >= 3.0
